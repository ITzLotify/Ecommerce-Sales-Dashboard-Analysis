# E-Commerce Sales Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data using Power BI.
The objective is to identify sales trends, profit patterns, and customer behavior.

## Tools Used
- Power BI
- DAX
- Excel / CSV

## Key Insights
- COD is the most used payment mode
- Maharashtra and Madhya Pradesh contribute the highest sales
- Clothing category dominates quantity but not always profit
- Profit varies significantly across sub-categories
